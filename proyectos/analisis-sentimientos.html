Resumen del Proyecto

Este es un experimento para calificar reseñas utilizando un modelo gratuito de análisis de sentimientos de Hugging Face. Solo debes ingresar una reseña y el sistema asignará una calificación del 1 al 5, donde 1 representa total disgusto o desacuerdo y 5 indica total acuerdo o excelencia. Como parte del proyecto, se emplea el analizador en línea disponible en la página de Hugging Face y, además, se desarrolla una aplicación de escritorio que guarda las reseñas en una base de datos SQL. El proyecto aún está en construcción.
